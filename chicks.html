Overview

A complete design process — nothing skipped

Chicks is a mobile application project where I ran the full product design cycle independently: from initial user research through final UI and design system documentation. The project was structured to demonstrate the complete discipline — not just visual output, but the thinking that leads to it.

Every major decision in the UI traces back to a research finding or a usability test result. This case study documents that traceability.

Approach

Research → Synthesis → Architecture → Wireframe → Test → Iterate → Final UI → Design System. Each stage informed the next.

01 — User Research

Understanding the user before defining the product

I conducted 10 semi-structured interviews with target users, followed by a competitive analysis of 6 apps in the same space. The research goal was to understand what the product needed to do, and what existing options were failing to do well.

04 — Usability Testing

Testing the prototype with real users

I ran two rounds of moderated usability tests using a clickable Figma prototype. Round 1 (6 participants) tested the overall navigation architecture. Round 2 (4 participants) validated the revised design after incorporating Round 1 findings.

The most significant change from testing: the original main navigation used a bottom tab bar with 5 items. Users consistently skipped the 4th and 5th tabs — they were consolidated into a single "More" entry, reducing the navigation to 4 tabs.

05 — Final UI

The product, fully designed

The final UI applied a visual system built around legibility, generous white space, and clear hierarchy. Every interactive element meets the 44px minimum touch target. Colors and contrast ratios meet WCAG AA for accessibility.

06 — Design System

Handoff-ready component library

The project concluded with a documented mobile design system: color tokens, type scale, spacing system, and a component library covering all the elements used in the app. The system was annotated for engineering handoff with exact values and behavior notes.
